# Coach4U Resource Rebranding Project

## Overview

25 files were uploaded to the `cathcoach4u-couples-uploads` branch. This document catalogues each file, identifies which are suitable for the client portal, flags duplicates, and provides a prioritised rebranding plan.

The client portal has 4 sections: **Relationships**, **Business**, **Leadership**, **Strengths**.
All 25 uploaded files relate to the **Relationships** section (couples coaching).

## Notes

- All rebranded resources use Coach4U branding: #003366 dark blue, Aptos font, Australian English
- Each resource gets Coach4U logo in header, standard footer with contact details
- Resources are self-contained HTML files in `resources/relationships/`
- Portal page (`portal.html`) needs updating to list each new resource
- The Connection Pulse Reports with client names should NOT go on the portal
- The `brand-resource` skill (`.claude/commands/brand-resource.md`) should be used for each conversion
- Binary files (DOCX, large PDFs) need content extracted before conversion to branded HTML
